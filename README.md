# Global Food Crisis Dashboard

A Streamlit dashboard for exploring global acute food insecurity from 2016 to 2025, built using the Global Report on Food Crises (GRFC) dataset.

The dashboard is my individual coursework for the Data Science Project Lifecycle module.
